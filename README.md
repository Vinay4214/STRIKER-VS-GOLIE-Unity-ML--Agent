# STRIKER-VS-GOLIE-Unity-ML--Agent
This soccer game is a Unity project using the ML-Agents toolkit. It is like a small robot soccer match where the players are controlled by AI instead of a person.  The game was created from the Unity ML-Agents “Soccer” example. Unity makes the 3D world, field, ball, and players. ML-Agents teaches the player robots how to play soccer using training.
